# donet CLI

Les commandes les plus utilisées:
- `dotnet new`
- `dotnet restore`
- `dotnet build`
- `dotnet test`
- `dotnet run`
- `dotnet publish`
- `dotnet pack`

Executer les commandes suivantes (pour plus tard):
- **`dotnet tool list --global`**
- **`dotnet tool install dotnet-ef --global`**
